# 연령별/시간대별 데이터 UI 및 지표 표 개선 (2026-06-26)

## 변경 내용
- 연령별데이터 표를 시간대별데이터와 동일하게 전체 주요 지표 컬럼으로 표시하도록 변경
  - 노출수, 클릭수, 전환수, ROAS(전환), 구매완료전환수, 구매완료 ROAS, 총비용, 평균노출순위
- 연령/성별 동시 집계 응답 시 연령대별 성별 분리 행과 연령대 합계 행을 함께 표시
- 성별 단독 집계가 있는 경우 `성별 전체 지표` 표를 별도로 노출
- 시간대별데이터 표도 동일 컴포넌트를 사용하도록 정리하고, 선택 중인 지표 컬럼을 강조 표시
- 연령별/시간대별 데이터 카드 UI를 더 깔끔하게 조정
  - 헤더/상태 배지/필터 영역/요약 카드/차트/표 섹션 스타일 개선
  - 표 가독성 향상을 위해 첫 번째 컬럼 고정, hover, 선택 지표 강조 적용
