# 시간대별 데이터 장기간 조회 처리 패치 (2026-06-26)

## 변경 내용
- 네이버 `/stats` `hh24` breakdown도 선택 기간을 30일 단위로 나눠 조회한 뒤 시간대별로 합산합니다.
- 최근 90일처럼 긴 기간은 `2026-04-03~2026-05-02`, `2026-05-03~2026-06-01`, `2026-06-02~2026-07-01` 형태의 구간 요청으로 처리합니다.
- 응답에 요청 기간(`requested_since`, `requested_until`)과 실제 조회 기간(`since`, `until`)을 동일하게 반환합니다.
- `debug.date_chunks`에서 실제 분할 조회 구간을 확인할 수 있습니다.

## 참고
- 긴 기간 조회는 단일 요청으로 보내지 않고 여러 `/stats` 요청을 합산하므로, 대상 광고그룹 수와 기간이 길수록 응답 시간이 늘어날 수 있습니다.
