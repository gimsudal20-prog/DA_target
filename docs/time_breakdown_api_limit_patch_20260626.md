# 시간대별 데이터 API 제한 처리 패치 (2026-06-26)

## 변경 내용
- 선택 기간 전체 합계는 네이버 `/stats`를 30일 단위로 나눠 조회한 뒤 합산합니다.
- 시간대별 `hh24` 상세 breakdown은 네이버 API가 최근 7일 이내 기간만 허용하므로, 오래된 구간을 반복 요청하지 않습니다.
- 차트와 표는 API 지원 범위의 실제 시간대 데이터로 표시하고, 상단 요약 카드는 선택 기간 전체 합계로 표시합니다.
- `debug.summary_chunks`에서 전체 합계 분할 조회 구간을, `debug.time_breakdown_range`에서 시간대 breakdown 조회 범위를 확인할 수 있습니다.

## 참고
- 네이버 검색광고 API가 제공하지 않는 과거 시간대별 상세 분포는 코드에서 실제값으로 복원할 수 없습니다.
