# 시간대별 데이터 API 제한 처리 패치 (2026-06-26)

## 변경 내용
- 네이버 `/stats` `hh24` breakdown 조회 시 최근 7일 초과 기간에서 발생하는 `11004 BAD_REQUEST` 오류가 화면에 반복 노출되지 않도록 처리했습니다.
- 시간대별 데이터는 API 지원 범위인 최근 7일(오늘 제외 기준)로 자동 보정해 조회합니다.
- 응답에 원래 요청 기간(`requested_since`, `requested_until`)과 실제 조회 기간(`since`, `until`)을 함께 반환합니다.
- `debug.range_was_clamped`로 기간 자동 보정 여부를 확인할 수 있습니다.

## 참고
- 네이버 검색광고 API의 시간대별(`hh24`) breakdown 자체가 최근 7일까지만 지원되어, 코드만으로 30일/90일 과거 시간대 breakdown을 직접 조회할 수는 없습니다.
