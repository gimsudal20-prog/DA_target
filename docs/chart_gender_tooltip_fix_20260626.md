# 연령별 성별 데이터/툴팁 커서 보정 패치 (2026-06-26)

## 반영 내용
- 차트 hover 시 마우스 커서가 십자가(crosshair)로 바뀌지 않도록 제거했습니다.
- 데이터가 있는 막대/포인트 위에서는 일반 클릭 포인터로만 보이게 조정했습니다.
- 연령별데이터 결과 화면에 성별 데이터를 다시 함께 노출했습니다.
- 성별 데이터는 연령별 차트 오른쪽 보조 패널에 표시되며, 선택 지표 기준 수치/비중/결과당 비용을 함께 보여줍니다.
- 성별 전체 지표 표를 연령별 전체 지표 표 아래에 추가했습니다.
- 남성/여성 차트 색상을 레퍼런스에 가깝게 보라/청록 계열로 정리했습니다.
- 성별 시리즈는 선택 지표 값이 0이어도 다른 지표에 데이터가 있으면 숨기지 않도록 보정했습니다.
