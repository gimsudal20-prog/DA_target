# 소재별 성과 이미지 저장 기능 수정 (2026-06-24)

## 수정 내용
- 이미지 저장 시 SVG `foreignObject` 캡처 대신 로컬 `html2canvas` 캡처를 사용하도록 변경해 PNG 저장 시 캔버스 보안 오염 오류를 피했습니다.
- `html2canvas`를 `static/vendor/html2canvas.min.js`에 로컬 정적 파일로 추가해 외부 네트워크 없이 저장 기능이 동작하도록 했습니다.
- PNG 다운로드 함수가 Promise를 반환하도록 수정해 저장 완료/실패 흐름을 안정화했습니다.
- 화면 캡처 방식이 실패할 경우 안정 모드 캔버스 저장으로 자동 재시도하도록 보완했습니다.
- 캡처 캔버스 크기를 최대 픽셀 기준으로 제한해 큰 조회 결과에서도 저장 실패 가능성을 줄였습니다.
- 기본 캡처 스케일을 1.5배 이상으로 조정해 저장 이미지의 글자 선명도를 높였습니다.
- 성과 없는 광고그룹 자동 제외 기준은 클릭·비용·전환·매출이 모두 0인 경우로 유지했습니다.
