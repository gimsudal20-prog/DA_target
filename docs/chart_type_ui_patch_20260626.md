# 연령별/시간대별 차트 보기 개선 패치

## 변경 내용
- 연령별데이터/시간대별데이터 결과 차트에 `차트 유형` 선택 드롭다운 추가
- 추천 보기, 막대형, 라인형, 누적 막대, 도넛 비중, 히트맵 지원
- 시간대별 추천 보기는 라인형으로 표시해 시간 흐름을 더 쉽게 확인
- 연령별 성별 데이터가 있는 경우 추천 보기는 누적 막대로 표시
- 차트 헤더와 선택 UI를 더 깔끔하게 정리
- 도넛/히트맵에서는 별도 범례 대신 캔버스 내부에 비중/강도 정보를 표시
- 연령별데이터 표는 시간대별데이터처럼 전체 지표 컬럼 기준으로 표시

## 적용 방법
프로젝트 루트에 압축을 풀어 `templates/index.html` 파일을 덮어쓰면 됩니다.
실행 파일은 재빌드하지 않았습니다.
