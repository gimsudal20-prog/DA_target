# 소재 조회/엑셀 설명 컬럼 보강 패치 (2026-04-30)

## 목적
계정 단위 소재 조회/엑셀 다운로드에서 `요약` 컬럼이 소재 제목 중심으로만 표시되어, 파워링크 기본소재의 설명 문구를 함께 확인하기 어려운 문제를 보완했습니다.

## 변경 사항
- 소재 수집 시 제목/설명/URL 필드를 별도 추출
  - `headline` → 소재 제목
  - `description` → 소재 설명
  - `pcFinalUrl` → PC 연결 URL
  - `mobileFinalUrl` → 모바일 연결 URL
  - `productName`, `referenceKey` 보조 필드
- 소재 엑셀 컬럼 추가
  - 소재 제목
  - 소재 설명
  - 상품명
  - PC 연결 URL
  - 모바일 연결 URL
  - 참조키
- 선택 광고그룹 상세 소재 조회 화면에도 `소재 제목`, `소재 설명`, `요약`을 분리 표시
- 기존 소재 ID/입찰가/상태/캠페인/광고그룹 컬럼은 유지

## 성능 영향
추가 API 호출은 없습니다. 기존 소재 조회 응답 안에 포함된 `ad.headline`, `ad.description`, URL 필드를 파싱해 컬럼만 확장했습니다.
