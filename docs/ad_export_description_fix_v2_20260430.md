# 소재 설명 엑셀 누락 수정 v2

## 확인된 증상
업로드된 `account_ads_account_20260430_103115.xlsx`의 헤더에는 `소재 제목`, `소재 설명` 컬럼이 없고 기존 `요약` 컬럼만 존재했다.
따라서 설명값이 비어 있는 문제가 아니라, 엑셀 생성 컬럼 세트가 구버전으로 실행되는 상태였다.

## 수정 내용
- `services/account_lookup_service.py`
  - 소재 엑셀 다운로드 컬럼에 `소재 제목`, `소재 설명`, `상품명`, `PC 연결 URL`, `모바일 연결 URL`, `참조키`를 확정 반영.
  - 캐시/구버전 rows에 대비해 `summary`가 `제목 / 설명` 형태일 경우 제목/설명을 보정.
- `app.py`
  - 소재 조회 원본 응답에서 설명 필드가 `ad.description` 외 nested `pc.description`, `mobile.description`, JSON 문자열 내부에 있어도 찾도록 보강.
  - 추가 API 호출 없이 기존 소재 조회 응답에서만 추출.

## 성능 영향
추가 API 호출 없음. 기존 응답 dict 내부 탐색만 추가되므로 다운로드 시간 증가가 거의 없다.

## 적용 후 기대 헤더
`캠페인유형, 캠페인명, 광고그룹유형, 광고그룹명, 소재유형, 상태, 소재 제목, 소재 설명, 상품명, PC 연결 URL, 모바일 연결 URL, 요약, ...`
