# 디자인 밀도 정리 패치 (2026-06-28)

## 목적
- 좌측/중앙 패널이 본문을 과하게 압박하는 문제 완화
- 상단 선택 상태 표시와 우하단 플로팅 선택 상태의 중복 제거
- 도움말/기능 검색 영역을 슬림하게 정리해 실제 작업 카드가 더 빨리 보이도록 조정
- 카드 내부 여백과 버튼 역할색을 정리해 외부 공유용 화면 완성도 개선

## 변경 사항
1. 레이아웃 폭 조정
   - 광고주 패널 기본 폭 축소
   - 캠페인/광고그룹 패널 기본 폭 축소
   - 1280px 이하 반응형 폭 별도 조정

2. 상단 영역 정리
   - 선택 상태는 브레드크럼 우측 pill에서만 강조
   - workspace toolbar의 선택 상태 텍스트 숨김
   - 도움말 허브는 기능 검색 중심으로 슬림화
   - 안내 영역이 접힌 상태에서는 즐겨찾기/최근 작업 패널도 숨김

3. 작업 영역 가독성 개선
   - 메인 탭 sticky 위치 재정리
   - 카드 내부 padding 확대
   - 카드 제목 영역 간격 재정리
   - 폼 그룹과 버튼 row 간격 보정

4. 버튼 역할색 정리
   - 조회/실행: 파랑
   - 다운로드/엑셀/내보내기: 초록
   - 삭제/위험 액션: 빨강
   - 보조 액션: 기존 neutral 톤 유지

5. 플로팅 위젯 정리
   - 중복 선택 상태 문구 제거
   - 선택된 항목이 있을 때만 변경/복사/삭제 빠른 작업 노출
   - 우하단 상단 이동 버튼은 유지
