# 확장소재 유형별 엑셀 내보내기 패치

## 목적
- 확장소재 엑셀 다운로드를 유형별로 받을 수 있도록 개선.
- 버튼을 여러 개 늘리지 않고 드롭다운 1개로 유형을 선택하도록 구성해 UI 복잡도 증가를 최소화.
- 이미지 서브링크/파워링크 이미지 계열은 이미지 ID를 더 확인하기 쉽게 `이미지 ID 수`, `이미지별 ID`, `이미지 ID 원문` 컬럼으로 분리.

## 반영
- `templates/index.html`
  - 계정 단위 조회 영역의 확장소재 엑셀 버튼 앞에 유형 선택 드롭다운 추가.
  - 기본값은 `확장소재 전체`.
  - 다운로드 payload에 `extension_type`만 추가.
  - 확장소재 미리보기에서 이미지 ID 수를 함께 표시.
- `services/account_lookup_service.py`
  - `/export_account_extensions_excel`에서 `extension_type`을 받아 행 필터링.
  - 기존 `/query_account_extensions` 결과 캐시를 그대로 활용하므로 유형별 다운로드를 위해 별도 API 수집 호출을 추가하지 않음.
  - `SUB_LINKS`이지만 이미지 ID가 있는 행은 `이미지 서브링크`로 분류해 내보내기 필터에 포함.
- `app.py`
  - 확장소재 엑셀 신규 컬럼 너비 추가.

## 성능 기준
- 유형별 다운로드 시 네이버 API를 유형별로 다시 여러 번 호출하지 않음.
- 기존 계정 확장소재 조회/엑셀 캐시 흐름 유지.
