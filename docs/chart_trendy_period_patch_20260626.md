# 차트/조회 기준/조회기간 개선 패치 (2026-06-26)

## 반영 내용
- 연령별데이터/시간대별데이터 광고 유형 조회 기준 순서를 파워링크 → 쇼핑검색 → 혼합검색으로 정렬했습니다.
- 기본 조회 유형을 파워링크로 변경했습니다.
- 조회기간 옵션에 최근 30일, 최근 90일을 추가했습니다.
- 오늘은 통계 데이터가 없다는 전제에 맞춰 연령별/시간대별 모두 오늘을 항상 제외하도록 변경했습니다.
- 시간대별데이터의 오늘 옵션을 제거했습니다.
- 직접 기간 선택 시 종료일 최대값을 항상 어제로 제한했습니다.
- 차트 패널, 차트 배경, 막대/라인/도넛/히트맵 색감과 그림자를 더 밝고 트렌디한 스타일로 개선했습니다.
- 툴팁 폭을 넓혀 이전보다 읽기 편하게 조정하고, 호버 데이터 표시는 유지했습니다.
