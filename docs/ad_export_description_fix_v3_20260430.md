# 소재 설명 엑셀 누락 수정 v3

업로드된 엑셀에서 `소재 제목`, `소재 설명` 헤더가 없는 것이 확인되어, 특정 라우트/레거시 경로가 구버전 컬럼 세트를 사용해도 `_build_asset_lookup_workbook` 단계에서 소재 엑셀 컬럼을 강제 보정하도록 수정했습니다.

- 소재 엑셀로 판단되는 컬럼 세트에 `소재 제목`, `소재 설명`, `상품명`, `PC 연결 URL`, `모바일 연결 URL` 자동 삽입
- 기존 `요약` 컬럼은 유지
- `headline/description`이 없으면 `summary` 또는 중첩 `ad` 객체에서 보정
- 엑셀 메타데이터에 `컬럼버전: ad-export-title-description-v3` 표시

적용 후 다운로드한 소재 엑셀 1~5행에 위 컬럼버전과 `소재 제목`, `소재 설명` 헤더가 보여야 합니다.
