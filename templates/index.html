개별 입찰가 설정으로 바꿀 때는 현재 적용 중인 유효 입찰가를 각 키워드/소재의 개별 입찰가로 저장합니다.
상/하한 기준 자동 증감
예: 상한 1,000원 이상이면 1,000원 기준 -100원 → 900원, 하한 300원 이하면 300원 기준 +100원 → 400원으로 변경됩니다. 쇼핑검색은 키워드가 아니라 광고그룹 내 소재 입찰가를 기준으로 처리합니다.
검색어 기준 파워링크 키워드 입찰가중치 변경
현재 계정의 파워링크 키워드 전체에서 검색어와 교집합이 있는 키워드를 찾아 입찰가중치를 일괄 변경합니다.
기본은 부분일치이며, 부분일치는 입력한 단어/토큰이 모두 포함된 키워드를 찾습니다. 필요할 때만 완전일치로 바꿔서 동일 키워드만 잡을 수 있습니다.
목표 평균순위 기준 입찰가 (자동 계산)
파워링크는 키워드 기준, 쇼핑검색은 소재 기준으로 목표 평균순위 입찰가를 자동 계산합니다.
시간대 및 가중치 일괄 설정
현재 선택한 시간대를 기준으로 기존 스케줄을 덮어씁니다.
구간별로 요일을 따로 지정하지 않으면 여기서 고른 요일을 공통으로 사용합니다.
예: 월~금 00~07시는 100%, 월~금 08~12시는 120%, 토요일만 별도 가중치처럼 요일별로 나눠서 적용할 수 있습니다. 종료 시각은 24시까지 선택할 수 있어요.
구간에 요일을 따로 지정하면 그 구간은 지정한 요일에만 적용됩니다. 지정하지 않으면 위 공통 요일을 사용합니다.
검색어 기준 파워링크 키워드 조회 / 엑셀 다운로드
현재 계정 전체 조회는 유지하면서, 필요하면 좌측에서 체크한 파워링크 캠페인 안에서만 검색할 수 있어요.
쉼표(,), 줄바꿈, 세미콜론(;)으로 여러 검색어를 한 번에 넣을 수 있어요. 예: 종로, 용산, 마포, 강남, 서초, 송파
포함 검색어에 일치하더라도 제외어에 걸리면 결과에서 제외합니다.
기본은 현재 계정 전체 파워링크 키워드에서 조회합니다.
부분일치: 각 검색어를 포함하는 키워드 조회 / 완전일치: 입력한 검색어와 정확히 같은 키워드만 조회
선택 캠페인 기준은 좌측 캠페인 체크 후, 선택 그룹 기준은 좌측 광고그룹 체크 후 조회하면 됩니다.
아직 조회한 결과가 없습니다.
검색어 기준 파워링크 키워드 입찰가 변경
현재 계정의 파워링크 키워드 전체 또는 좌측에서 체크한 파워링크 캠페인 안에서 검색어와 교집합이 있는 키워드를 찾아 입찰가를 일괄 변경합니다.
기본은 부분일치이며, 부분일치는 입력한 단어/토큰이 모두 포함된 키워드를 찾습니다. 필요할 때만 완전일치로 바꿔서 동일 키워드만 잡을 수 있습니다.
파워링크 키워드 개수 조회
현재 계정의 파워링크 총 등록 키워드 수와, 좌측에서 체크한 파워링크 캠페인의 키워드 수를 함께 조회합니다.
캠페인을 체크하지 않아도 전체 계정 기준 총 개수는 조회됩니다.
아직 조회한 결과가 없습니다.
파워링크 캠페인 내부 중복키워드 조회
체크한 캠페인 중 파워링크 캠페인만 대상으로, 서로 다른 광고그룹 2개 이상에 들어간 동일 키워드를 찾아줍니다.
아직 조회한 결과가 없습니다.
캠페인 전체 복사
직접 지정 이름을 입력하면 접미사 대신 사용됩니다.
입력 개수는 선택 캠페인 수 × 복사 개수와 같아야 합니다. (상한 없음)
직접 지정 이름이 없으면 복사 개수가 2개 이상일 때 원본명 기준으로 자동 번호가 붙습니다. 예: 경쟁사, 경쟁사2, 경쟁사3
광고그룹 전체 복사
직접 지정 이름을 입력하면 접미사 대신 사용됩니다.
입력 개수는 선택 광고그룹 수 × 복사 개수와 같아야 합니다. 체크한 캠페인 하위 전체 복사도 같은 기준으로 포함됩니다. (상한 없음)
직접 지정 이름이 없으면 복사 개수가 2개 이상일 때 원본명 기준으로 자동 번호가 붙습니다. 예: 광고그룹, 광고그룹2, 광고그룹3
광고그룹명 일괄 변경
현재 좌측 광고그룹 목록 순서 기준으로 적용됩니다.
선택한 광고그룹 수와 입력한 이름 수가 같아야 하며, 같은 캠페인 안에서 기존 이름과 겹치면 차단됩니다.

연령대/성별/지역 등 추가 타겟과 요일/시간대를 선택 캠페인 또는 선택 광고그룹 전체에 복사 적용합니다.
