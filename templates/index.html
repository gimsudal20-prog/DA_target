예: 00~06시는 50%, 07~24시는 100%처럼 구간마다 다른 입찰가중치를 적용할 수 있습니다. 화면의 종료 시각은 다음 시각 기준이라 00~06시를 적용하려면 00시~07시로 지정됩니다.

마지막으로 선택한 시각은 종료 시각으로 처리합니다. 예: 00·01·02·03 선택 → 00~03, 07~23 선택 → 07~24

타겟 설정 일괄 적용
원본 광고그룹은 아래 드롭다운에서 별도로 선택합니다. 좌측 체크박스는 적용 대상 선택에만 사용됩니다.
원본 광고그룹의 연령대/성별/지역/요일·시간대 타겟을 선택 캠페인 또는 선택 광고그룹 전체에 복사 적용합니다.
이 그룹의 타겟팅 설정을 기준으로 복사합니다. 적용 대상 체크박스와 분리되어 있습니다.
둘 다 체크하면 선택 광고그룹과 선택 캠페인 하위 그룹에 모두 적용됩니다. 원본 광고그룹은 적용 대상에서 자동 제외됩니다.
연령대=AG, 성별=GN, 지역=RL/RP, 시간대=SD 기준으로 적용됩니다.
캠페인/광고그룹 삭제 관리
캠페인 삭제와 광고그룹 삭제는 하위 데이터 일괄삭제와 분리했습니다. 좌측 체크박스에서 삭제할 대상을 선택한 뒤 아래 버튼으로 실행하세요.
좌측에서 체크한 캠페인 자체를 삭제합니다. 광고그룹 체크는 이 작업에서 제외됩니다.

하위 데이터 일괄 삭제 (위험)
좌측 체크박스에서 선택한 캠페인 또는 광고그룹 하위의 키워드·소재·확장소재 데이터를 일괄 삭제합니다. 캠페인/광고그룹 자체 삭제는 위의 별도 관리 영역에서 실행합니다.
둘 다 체크하면 선택 캠페인 하위 그룹과 선택 광고그룹을 합쳐서 삭제 대상을 찾습니다.
예: 추가제목과 이미지형 서브링크만 체크하면 해당 유형만 삭제합니다. 전체 삭제가 체크되어 있으면 다른 유형 선택은 무시됩니다.
파워링크 소재ID 엑셀 대량삭제
엑셀 파일의 소재ID 컬럼에 입력된 파워링크 소재를 한 번에 삭제합니다. 좌측 캠페인/그룹 선택과 관계없이 파일에 있는 소재ID만 삭제합니다.
권장 컬럼명: 소재ID / nccAdId / 광고ID. 첫 번째 시트 기준으로 처리합니다.
CSV 템플릿은 엑셀에서 바로 열 수 있습니다. 소재ID만 붙여넣어 사용하세요.
확장소재ID 엑셀 대량삭제
엑셀 파일의 확장소재ID 컬럼에 입력된 확장소재를 한 번에 삭제합니다. 추가설명, 서브링크, 홍보문구, 이미지형 서브링크 모두 ID 기준으로 처리합니다.
권장 컬럼명: 확장소재ID / adExtensionId / nccAdExtensionId. 첫 번째 시트 기준으로 처리합니다.
계정 확장소재 엑셀에서 확장소재 ID만 붙여넣어 사용하세요.
ID별 소재/확장소재 상태변경
삭제하지 않고 소재ID/확장소재ID 기준으로 ON/OFF만 변경할 때 사용합니다. 줄바꿈, 쉼표, 엑셀에서 복사한 한 열 붙여넣기를 지원합니다.
